Getting Started with n8n

Accessing n8n

1 Open n8n in your browser
  • If self-hosted: Navigate to http://localhost:5678 (default port)
  • If cloud: Log in to n8n.io
2 Sign in or create account
  • Click "Sign In" button in the top right corner
  • Enter your credentials or create a new account
3 Navigate to Workflows
Click: Workflows → (in left sidebar)

You'll see a list of your existing workflows or an empty workspace if you're new.

Creating a New Workflow

Initial Setup

1 Create New Workflow
Click: "New Workflow" button (top right, or "+" icon)

A blank canvas will appear with a "Start" node already placed.

2 Name Your Workflow
Click: Workflow name (top center, "Untitled Workflow") → Type: "PDF Org Chart Processor"
3 Save Your Workflow
Click: "Save" button (top right) or press Ctrl+S (Windows) / Cmd+S (Mac)

Adding Nodes to Your Workflow

Method 1: Using the Node Panel

1 Open Node Panel
Click: "+" button (left side of canvas) or press Space key

A searchable panel with all available nodes will appear.

2 Search for Node
Type in search box: "Webhook" or "HTTP Request"

Nodes matching your search will appear below.

3 Add Node to Canvas
Click: Node name (e.g., "Webhook")

The node will appear on your canvas. You can drag it to reposition.

Method 2: Connecting from Existing Node

1 Hover over Node
Hover mouse over: Output dot (right side of a node)

The output dot will highlight.

2 Click and Drag
Click and drag: From output dot → To empty space

A connection line will appear, and the node panel will open automatically.

3 Select Next Node
Click: Desired node from the panel

The new node will be connected automatically to the previous one.

Configuring Nodes

Opening Node Configuration

1 Select Node
Click: Node on canvas

The node will be highlighted, and a configuration panel will appear on the right side.

2 View Configuration Panel

The right panel shows:

  • Parameters tab: Main configuration options
  • Settings tab: Node-specific settings
  • Credentials section: For API keys and authentication

Common Configuration Tasks

1 Rename Node
Double-click: Node name on canvas → Type new name

Or use the "Name" field in the configuration panel.

2 Add Credentials
Click: "Create New Credential" → Select credential type → Enter API key

Credentials are stored securely and can be reused across workflows.

3 Use Expression Editor
Click: "Expression" button (next to input fields) → Enter JavaScript expression

Use {{ $json.fieldName }} to reference data from previous nodes.

Connecting Nodes

Manual Connection

1 Select Source Node
Click: Output dot (right side of source node)
2 Drag to Target Node
Drag: From output dot → To input dot (left side of target node)

A connection line will appear between the nodes.

3 Verify Connection

The connection line should be solid. If dashed, the connection may be incomplete.

Disconnecting Nodes

1 Select Connection
Click: Connection line between nodes
2 Delete Connection
Press: Delete key or right-click → "Delete"

Testing Your Workflow

Manual Execution

1 Execute Workflow
Click: "Execute Workflow" button (top right) or press Ctrl+Enter

The workflow will run from the Start node.

2 View Execution Results
Click: Node after execution → View "Output" tab in right panel

You'll see the data that passed through each node.

3 Test Individual Nodes
Right-click: Node → "Execute Node"

This runs only the selected node with data from previous nodes.

Using Test Data

1 Open Node
Click: Node → "Test Step" tab
2 Enter Test Data
Paste or type: JSON test data in the test data field
3 Run Test
Click: "Test Step" button

The node will execute with your test data.

Saving and Activating Workflows

Saving Workflows

1 Save Workflow
Click: "Save" button (top right) or Ctrl+S / Cmd+S

Your workflow is saved but not active yet.

2 Auto-save

n8n auto-saves your work periodically. Look for the "Saved" indicator in the top bar.

Activating Workflows (Production Mode)

1 Toggle Active Switch
Click: "Active" toggle switch (top right, next to Save)

The switch will turn green when active.

2 Confirm Activation

Active workflows will:

  • Respond to webhook triggers automatically
  • Run on schedule (if using Schedule Trigger)
  • Execute when manually triggered

💡 Tip: Only activate workflows when you're ready for production. Test thoroughly first!

Viewing Workflow Executions

Execution History

1 Open Executions
Click: "Executions" tab (top of workflow canvas)

You'll see a list of all workflow executions.

2 View Execution Details
Click: Execution entry in the list

You'll see:

  • Execution status (Success/Error)
  • Execution time
  • Data at each node

3 Re-run Execution
Click: "Re-run" button (in execution details)

This re-executes the workflow with the same input data.

Keyboard Shortcuts

Essential Shortcuts

  • Space - Open node panel
  • Ctrl+S / Cmd+S - Save workflow
  • Ctrl+Enter / Cmd+Enter - Execute workflow
  • Delete - Delete selected node or connection
  • Ctrl+Z / Cmd+Z - Undo
  • Ctrl+Y / Cmd+Shift+Z - Redo
  • Ctrl+F / Cmd+F - Search nodes
  • Esc - Close panels

Tips and Tricks

💡 Tip 1: Use the Expression Editor

Click the "Expression" button next to any field to write JavaScript expressions. Use {{ $json.fieldName }} to access data from previous nodes.

💡 Tip 2: Node Templates

Right-click a node → "Save as Template" to reuse node configurations across workflows.

💡 Tip 3: Zoom and Pan

Use mouse wheel to zoom, and click + drag on empty canvas to pan around large workflows.

⚠️ Warning: Always Test First

Before activating a workflow, test it thoroughly with sample data. Use the "Test Step" feature to debug individual nodes.
